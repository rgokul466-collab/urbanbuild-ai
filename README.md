# UrbanBuild AI — Site Platform (prototype)

A role-based web app for UrbanBuild AI, an Australian construction PropTech company.
Employees sign in (Site Worker, Supervisor, Safety Officer, Project Manager, Logistics, Admin)
and manage identity & access, site check-in, work packages, hazard reporting, credentials,
RBAC and audit — built from the team's Sprint 1 use cases and product backlog.

Static site (HTML/CSS/JS, no build step). Open `index.html` or visit the published GitHub Pages link.
Prototype only — data is mock and stored per-browser (localStorage); there is no shared backend.
